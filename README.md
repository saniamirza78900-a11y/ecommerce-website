# 🛒 AL Amaanat E-commerce Website

## 📌 Project Overview
This is a simple E-commerce Website developed as a Web Development Major Project.  
It allows users to browse products like dry fruits and dates, add them to cart, and place orders.

---

## 🚀 Features

### 👤 User Module
- Registration & Login
- View Products
- Add to Cart
- Place Order (Cash on Delivery)
- Order Notification

### 🛠️ Admin Module
- Admin Login
- Add Products
- View Orders
- View Customers
